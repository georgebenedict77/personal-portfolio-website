# Personal Portfolio Website

A responsive personal portfolio website for George (`benjoji77`), a Computer Science student and graphic designer from Kenya. The site presents software work, design samples, a CV, and direct contact paths in a polished, recruiter-friendly format.

## Live site

[georgebenedict77.github.io/personal-portfolio-website](https://georgebenedict77.github.io/personal-portfolio-website/)

## Highlights

- Strong personal-brand landing page with clear positioning
- Selected work section covering software and design projects
- Dedicated CV page and supporting design sample pages
- Direct contact area with email shortcuts and quick outreach flow
- GitHub Pages deployment for simple public hosting

## New Project Added

- **Notepad Desktop App (Java Swing)**
- One-class ActionListener methodology with real notepad workflow
- Features include file handling, editing tools, find/replace, word wrap, and status bar
- Source branch: [notepad-desktop-app](https://github.com/georgebenedict77/personal-portfolio-website/tree/notepad-desktop-app)
