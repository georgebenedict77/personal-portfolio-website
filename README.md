# Personal Portfolio Website

A responsive personal portfolio website for George (`benjoji77`), a Computer Science student and graphic designer from Kenya.

## What this project includes

- A polished landing page with professional positioning
- A selected work section for software and design projects
- A dedicated CV page and design sample pages
- A direct contact section with quick email actions

## Built with

- HTML
- CSS
- JavaScript
